## `>> THE COMMONS PHILOSOPHY`

Code that lives in a vault helps nobody. Code that lives in **the commons** - **reviewed, forked, extended, and reused -** builds whole ecosystems.

CodeCommons is an open ground where:
- **every student** can contribute without fear of judgment
- **every idea** gets a PR, not a rejection
- **every project** is a **shared asset**, not a solo trophy
